이제 전화·카톡 대신 여기에 올려요
예전에는 전화나 카톡으로 하나씩 보고했지만, 새로운 "시스템"으로 통합했어요.
보고 후, 승인여부까지 간편하게 확인가능해요. 📱
⚠️ 단, 경조휴가(결혼식·장례식 등)는 매장관리부에 직접 전화나 카톡으로 알려주세요.
2
모든 항목은 자세하게 작성해 주세요
기록을 남기실 때는 사유를 구체적으로 작성해 주세요.
(예: "몸살감기로 병원 방문" ✅ / "개인사정" ❌)
🔒 작성하신 사유는 관리자만 확인 가능하며, 목록에는 '개인사정'으로 표시됩니다.
3
유저페이에도 꼭 써주세요
모든 항목(지각 제외)은 "유저페이"에도 반드시 작성해 주세요.
휴가 기록 항목은 [승인] 상태로 변경된 후 유저페이에 등록해 주세요.
※ '대기중' 또는 '반려' 상태는 승인된 연차로 인정되지 않으므로, 승인 완료 후 등록해 주세요.

(로그인 아이디·비밀번호를 모르시면 → 인사회계부에 문의)
